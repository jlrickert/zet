# Karble cheat sheet

| Card  | Description           |
| ----- | --------------------- |
| Joker | Move out or pass turn |
| Ace   | Move out or one space |
| Four  | Reverse               |
| Seven | Split 7 moves         |
| Jack  | Swap or move 11       |
| Queen | Moves 12              |
| king  | Move out or moves 13  |
