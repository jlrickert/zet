# WOW Dragonflight season 4

Dragonflight season 4 is released on Tuesday April 23rd (2024-04-23).

My main was a rogue

## Mythic + dungeon list

- The Azure Vault
- Halls of infusion
- Neltharus
- Algeth'ar Academy
- Uldaman: Legacy of Tyr
- Brackenhide Hollow
- Ruby Life Pools
- The Nokhud Offensive

## New features

- Awakened raids

  Every week an old Dragonflight raid will drop improved rewards.
