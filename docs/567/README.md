# VLAN overview

(Virtual area network)

Separates local area network traffic

- trunk port

  Network port that routes traffic for multiple vlans

- tagged vlan

  Network port aware of port assigned too. A device connected needs to know what VLAN it is a part off.

- untagged vlan

  The default. Device assigned a vlan when plugged in by the port it is plugged into.

- tagged vs untagged vs Trunk port
- vlan tags and id

  ID header on frame

- Access port
- Trunk port
- bridge
- switch

- router

  Connects LAN's

- client isolation

  Prevents devices from connecting to other devices other than the internet

- port mirroring

  Allows for networks sniffing.
