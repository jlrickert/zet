# What is Multi-Factor authentication

Multi-Factor authentication is when 2 more pieces of evidence is use to
authenticate. Each pieces of evidence should be done by different mechanisms.
Some possible mechanisms include knowledge, possession, and inherence.

An examples of knowledge would be a password. Examples of possession would be
things like a phone, key, or physical device. Examples of inherence would be
FaceID or finger print.

Common MFA providers include Okta, Duo, LastPass, OneLogin, and Auth0.
