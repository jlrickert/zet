# Tiling window management on macOS with Yabai

Dump

```bash
brew install koekeishiya/formulae/yabai
brew install koekeishiya/formulae/skhd
```

```bash
cp /opt/homebrew/opt/skhd/share/skhd/examples/skhdrc ~/.skhdrc
```

```bash
skhd --start-service
skhd --stop-service
```
