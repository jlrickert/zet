# Bake: large 80/20 sourdough bowl

A tweak on the [labor day 2024 bread bowl](../827).

![bread bowl](./bread-bowl.jpg)

| Ingredient                  | Amount | Bakers Percentage |
| --------------------------- | ------ | ----------------- |
| [All purpose flour](../630) | 558g   | 80%               |
| [Turkey red](../739)        | 140g   | 20%               |
